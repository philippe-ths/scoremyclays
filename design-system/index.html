ScoreMyClays MVP Design System

Interactive prototypes demonstrating the ultra-simplified design system for clay pigeon shooting scoring. Each prototype showcases core MVP functionality with outdoor-optimized UI components.

MVP Design Principles

Ultra-Simplicity

One primary action per screen, extreme visual clarity, instant feedback under 100ms

Outdoor Optimized

High contrast for bright sunlight, large touch targets for gloved hands

Offline First

Core scoring functionality works perfectly without internet connection

Safety Conscious

Never distract from shooting activities, respect traditional clay shooting terminology
